# NodeJs
Node API Tutorial

## Instruction

- Install all dependencies
```shell
$ npm install
```

- Make a file nodemon.json that contains
```json
{
  "env": {
    "MONGO_ATLAS_PW": "YOUR MONGO PASSWORD",
    "JWT_KEY": "YOUR TOKEN PASSWORD"
  }
}
```

- Run Server
```shell
$ npm start
```

If you want to modify I recomend using

```shell
$ npm test
```
